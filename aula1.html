Módulo 01: Fundamentos JS

1. O que é JS?

O JavaScript é a linguagem que dá vida aos sites. Com ele, criamos botões que funcionam e sistemas inteligentes.

2. Variáveis (As Caixas)

Usamos variáveis para guardar nomes ou números. Exemplo: let player = "Minedev";

3. Lógica de Decisão

É o famoso "Se acontecer isso, faça aquilo". Usamos o IF para o computador decidir o que fazer.
