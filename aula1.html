Módulo 01

Fundamentos e Lógica de Programação

1. O que é JavaScript?

O JS é o "cérebro" do site. Enquanto o HTML cria os ossos, o JS cria as ações, como botões que funcionam e cálculos.

2. Variáveis

Imagine variáveis como caixas onde guardamos informações. Usamos let para criar uma:

let nome = "MineDevs";

3. Lógica de Decisão

O computador decide coisas usando o IF (Se).

Exemplo: Se a bateria for menor que 10%, mostre um aviso.
